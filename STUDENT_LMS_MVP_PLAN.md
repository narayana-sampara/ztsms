Act as a senior product manager, UI/UX designer, software architect, and full-stack developer.

I want to build an MVP Student Learning Management System for a single tutoring center.

The tutoring center teaches students from:
- CBSC

The MVP has four roles:

1. Admin
Admin should be able to:
- Login
- View dashboard
- Create tutors
- Edit tutors
- Activate/deactivate tutors
- View high-level curriculum, student, parent, and assignment activity

2. Tutor
Tutor should be able to:
- Login
- Create student profiles
- Create parent profiles
- Tag/link parents to students
- Link one or more parents/guardians to a student
- Link one parent to multiple students if required
- Assign curriculum to students
- Create curriculum master data
- Use curriculum references from recognized board websites by storing source URLs or uploaded reference documents
- Create grade/level, subject, unit, topic, and learning outcomes
- Upload resources
- Upload practice worksheets
- Create assignments
- Assign assignments to students
- Review student submissions
- Update student progress
- Add tutor feedback visible to parents

3. Student
Student should be able to:
- Login
- View assigned curriculum
- View subjects, units, topics, and learning outcomes
- View/download resources
- View/download practice worksheets
- View assignments
- Submit assignments
- View profile
- View learning progress

4. Parent
Parent should be able to:
- Login
- View linked child/children
- Track curriculum progress
- Track pending assignments
- Track overdue assignments
- View completed assignments
- View tutor feedback

Design the MVP as a modern EdTech SaaS web application.

Use this design style:
- Clean modern dashboard
- Responsive layout
- Sidebar navigation on desktop
- Mobile-friendly navigation
- Card-based UI
- Soft shadows
- Rounded corners
- Progress bars
- Status badges
- Curriculum tree view
- Resource cards
- Assignment cards
- Parent-student mapping screen
- Parent progress analytics
- Minimal visual clutter
- Student-friendly but professional

Frontend:

ASP.NET Core Web API, Entity Framework Core, JWT Authentication, Role-based Authorization, PostgreSQL

AWS S3
